<objective>
Provide parents with a high-level view of their child's learning journey. This involves read-only access to mastery scores, session summaries, and personalized learning recommendations.
</objective>

## Task 1: Parent Dashboard UI Layout
- **read_first**: `apps/frontend/src/components/dashboard/`
- **action**: Create a new `ParentDashboard` that shows one or more children's progress.
- **acceptance_criteria**:
  - Displays child's name, grade, and current subjects.
  - Highlights overall progress (e.g., 75% Math Mastery).

## Task 2: Session Summaries for Parents
- **read_first**: `packages/shared/src/tutoring/session.ts`
- **action**: Implement a "Weekly Report" view showing concepts learned and areas needing practice.
- **acceptance_criteria**:
  - Summarizes tutoring sessions into simple Thai sentences.
  - Shows time spent and engagement metrics.

## Task 3: Learning Recommendations
- **read_first**: `packages/shared/src/mastery/difficulty.ts`
- **action**: Implement a "Parent Recommendation" component suggesting topics for home practice.
- **acceptance_criteria**:
  - Matches the student's current learning path.
  - Provides parent-friendly explanations of the concepts.

## Verification Steps

1. Log in as a parent and view a child's dashboard.
2. Verify mastery data matches the student's real progress.
3. Check the "Weekly Report" for clarity and accuracy.
