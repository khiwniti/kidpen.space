<objective>
Build a student-facing dashboard that visualizes learning progress based on the pyBKT mastery tracking system. The UI should use Thai-localized metaphors (like Stars or Progress Rings) to encourage a growth mindset.
</objective>

## Task 1: Mastery Dashboard UI Design
- **read_first**: `apps/frontend/src/components/dashboard/`
- **action**: Create a new `MasteryDashboard` component that shows subjects and topics.
- **acceptance_criteria**:
  - Displays "คณิตศาสตร์" and "วิทยาศาสตร์" as main categories.
  - Shows progress for each topic (e.g., 3/5 concepts mastered).

## Task 2: Visual Progress Indicators (Stars/Rings)
- **read_first**: `packages/shared/src/mastery/types.ts`
- **action**: Implement a `MasteryStar` or `MasteryProgress` component that maps pyBKT probability to a visual state (0-5 stars).
- **acceptance_criteria**:
  - Visuals reflect real-time pyBKT data from `MasteryProvider`.
  - Color scheme is encouraging (not punitive).

## Task 3: Concept Tree Visualization
- **read_first**: `packages/shared/src/mastery/knowledge.ts`
- **action**: Implement a hierarchical view of concepts within a topic.
- **acceptance_criteria**:
  - Shows prerequisites and current learning path.
  - Highlights the next recommended concept to learn.
