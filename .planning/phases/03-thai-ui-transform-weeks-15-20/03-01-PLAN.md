<objective>
Complete the Thai localization of the entire student-facing platform. This involves translating all UI strings, ensuring STEM terminology is accurate and curriculum-aligned, and removing all English fragments from the core learning paths.
</objective>

## Task 1: Comprehensive Thai Translation Audit
- **read_first**: `apps/frontend/translations/th.json`, `apps/frontend/src/`
- **action**: Scan all components for hardcoded English strings and move them to `th.json`.
- **acceptance_criteria**:

## Task 2: IPST-Aligned STEM Terminology Integration
- **read_first**: Thai curriculum standards (IPST)
- **action**: Update STEM-related UI labels and tooltips to use formal Thai educational terms.
- **acceptance_criteria**:
  - Terms like "Mastery", "Concept", "Knowledge Component" have appropriate Thai equivalents.
  - Math/Science terms are consistent with Thai textbooks.

## Task 3: Cultural Context Adaptation
- **read_first**: `packages/shared/src/tutoring/prompts.ts`
- **action**: Update Socratic prompts and example problems to use Thai names, places, and metric units.
- **acceptance_criteria**:
  - AI tutor uses polite particles (ครับ/ค่ะ) correctly.
  - Word problems use context relevant to Thai students.

## Task 4: RTL and Font Rendering Optimization
- **read_first**: `apps/frontend/src/app/globals.css`
- **action**: Optimize Sarabun font loading and ensure line-height is appropriate for Thai script.
- **acceptance_criteria**:
