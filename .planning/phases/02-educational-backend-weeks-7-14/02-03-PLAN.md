<objective>
Build the Socratic tutoring engine that guides students through problems without revealing answers. Implements Thai-language encouragement patterns, misconception detection, and subject-specific prompt templates.
</objective>

## Task 1: Design Thai Socratic system prompts
- **read_first**: `backend/core/agents/` for existing agent patterns
- **action**: Create `backend/tutoring/prompts/socratic-system.ts` with:
  - Thai tutor persona: ครูพี่เลี้ยง (supportive tutor, not authority)
  - Core directive: ถามนำ (guiding questions, never give direct answers)
  - Face-saving encouragement on mistakes
  - Thai cultural context integration
- **acceptance_criteria**:
  - System prompt enforces Socratic behavior
  - Includes Thai-specific encouragement phrases
  - Explicitly forbids direct answer revelation
  - Culturally appropriate tone

## Task 2: Implement dialogue policy engine
- **read_first**: Task 1 output
- **action**: Create `backend/tutoring/policies/dialogue-policy.ts` with:
  - `neverGiveAnswer` policy: Block direct answer responses
  - `scaffoldHint` policy: Progressive hint generation
  - `misconceptionProbe` policy: Detect and address misconceptions
  - `encourageOnMistake` policy: Thai encouragement patterns
- **acceptance_criteria**:
  - Each policy is a pure function
  - Policies can be composed/chained
  - Returns modified response or null (pass-through)
  - Logs policy application for debugging

## Task 3: Create subject-specific prompt variants
- **read_first**: `backend/tutoring/prompts/socratic-system.ts`

  - Each includes subject-specific misconception patterns
- **acceptance_criteria**:
  - Each subject has tailored Socratic approach
  - Includes common misconception detection

  - Grade-appropriate difficulty scaling

## Task 4: Build misconception detection module
- **read_first**: `backend/tutoring/policies/`
- **action**: Create `backend/tutoring/misconceptions/detector.ts` with:
  - Pattern-based misconception detection
  - Subject-specific misconception library
  - Severity classification (minor/major)
  - Suggested probing questions
- **acceptance_criteria**:
  - Detects common math misconceptions (e.g., fraction operations)
  - Detects science misconceptions (e.g., force/motion)
  - Returns misconception type and suggested response
  - Extensible for new misconception patterns
